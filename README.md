# Churn Predictor Model

## Dataset
Il progetto utilizza il **Telco Customer Churn** dataset.
- **Fonte**: [Kaggle - Blastchar](https://www.kaggle.com/datasets/blastchar/telco-customer-churn)
- **Descrizione**: Dati relativi ai clienti di una compagnia telefonica, inclusi servizi sottoscritti e stato del churn.
- **File**: `data/raw/telco_churn.csv`
